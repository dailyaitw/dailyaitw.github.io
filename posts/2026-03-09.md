# 📅 3 月 9 日（週一）

### 🧠 2026/03/09 AI 重點
1. **Microsoft 發表「Wave 3」Copilot 重大更新**：推出 [Copilot Cowork](https://www.microsoft.com/en-us/microsoft-365/blog/2026/03/09/copilot-cowork-a-new-way-of-getting-work-done/)，可自動執行跨 Microsoft 365 的任務——自動建立簡報、拉取 Excel 數據、寄信約會議。
2. **Anthropic Claude 正式進駐 Microsoft 365**：Claude 成為 [Copilot 聊天中的可選模型](https://www.pcworld.com/article/3082700/move-over-copilot-claude-is-coming-to-microsoft-365.html)，Microsoft 走向「多模型策略」，不再只押注 OpenAI。
3. **Microsoft 365 E7 Frontier Suite 發表**：定價每人每月 $99，[5 月 1 日正式上市](https://blogs.microsoft.com/blog/2026/03/09/introducing-the-first-frontier-suite-built-on-intelligence-trust/)，另推出 Agent 365（$15/人/月）。
4. **Copilot Cowork 進入受限預覽**：目前少量企業客戶測試中，預計 3 月底透過 Frontier 計畫擴大開放。

### 💰 投資動向
- Microsoft 加碼 Anthropic 合作，將 Claude 嵌入其旗艦產品線，顯示 Anthropic 的商業化正式進入「平台級」階段。

### ⚙️ 技術突破
- Copilot Cowork 代表 AI Agent 從「對話助手」正式升級為「任務執行者」，可跨應用自動完成多步驟工作流。

### 🌍 社會影響
- 企業辦公模式面臨根本性轉變：AI 不再只是建議，而是直接代替員工執行操作性工作。知識工作者角色將從「執行者」轉向「審核者」。

### 📊 我的觀點
Microsoft 引入 Claude 是一步精明棋——既降低對 OpenAI 的依賴風險，又為企業客戶提供模型選擇的靈活性。E7 套件定價 $99/月看似大膽，但若 Agent 真能取代半個助理的工作量，ROI 其實很高。這是 AI SaaS 進入「溢價時代」的信號。
