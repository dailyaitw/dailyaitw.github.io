# 📅 3 月 3 日（週二）

### 🧠 2026/03/03 AI 重點
1. **DeepSeek V4 發布** — 1 兆參數的多模態模型，但每個 token 僅使用 320 億活躍參數，展現極致效率。配合中國「兩會」時程發布。
2. **Google Gemini 3.1 Flash-Lite 發布** — 走速度與成本效率路線，針對高流量工作負載優化，與 OpenAI 的「大而全」策略形成對比。
3. **OpenClaw 在 GitHub 突破 250,829 顆星** — 超越 React（243,000 星）與 Linux（218,000 星），成為 GitHub 史上第二高星數專案。
4. **MWC 2026 開幕（巴塞隆納，3/3-6）** — AI 眼鏡成為焦點，阿里巴巴/Qwen、三星、聯發科、科大訊飛等紛紛展示產品。
5. **#QuitGPT 抗議者聚集在 OpenAI 舊金山總部外** — OpenAI 與五角大樓的合作引發持續反彈。

### 💰 投資動向
- DeepSeek V4 的效率模式（1T 參數 / 32B 活躍）可能重塑推論成本結構，對 GPU 需求預測產生影響。

### ⚙️ 技術突破
- DeepSeek V4 的稀疏激活架構是重大突破：1 兆參數的智慧只需 32B 的成本運行，這是「以少勝多」理念的極致體現。
- Gemini 3.1 Flash-Lite 專注推論效率，代表 Google 在「平民化 AI」方向的持續投入。

### 🌍 社會影響
- OpenClaw 的開源成功（60 天超越 React 十年紀錄）證明社群驅動的 AI 工具有巨大潛力。OpenAI 贊助但不擁有程式碼，MIT 授權，社群治理 — 這是開源 AI 的新典範。

### 📊 我的觀點
DeepSeek V4 再次證明中國在 AI 效率工程上的實力。稀疏激活讓 1T 參數模型可以用極低成本部署，這對整個推論基礎設施產業有深遠影響。而 OpenClaw 的崛起顯示，未來 AI 的競爭不只是模型本身，更是生態系統與社群的競爭。
